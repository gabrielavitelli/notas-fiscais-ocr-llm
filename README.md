# notas-fiscais-ocr-llm
App para extração estruturada de dados de notas fiscais (PDF/imagem): OCR com DocTR, classificação com LLM (Groq ou Hugging Face), interface Streamlit. Extrai número, data, empresa, CNPJ, valor, moeda, rubrica, pesquisador. Exporta CSV, evita duplicatas, área de revisão. Roda no seu PC ou na nuvem (Streamlit Community Cloud).
